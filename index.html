1. Candlestick Pattern

是否出现明确的蜡烛触发，例如强阳、强阴、锤头、吞没或拒绝信号。

2. Chart Pattern

是否存在清晰结构，例如通道、三角、双顶双底、长方形、突破或回踩。

3. Wave

波浪阶段是否清晰，方向是否与当前想做的交易一致。

4. MA

均线是否支持当前方向，例如价位站上均线做多，或跌破均线做空。

5. Bollinger Band

布林带位置是否合理，例如上轨做空、下轨做多、中轨确认或带宽配合。

6. Fibo

是否位于关键 Fibo 区，例如 38.2 / 50 / 61.8 / 76.4 等有效位置。

高级过滤区（v2）过滤垃圾信号、加入时段与波动审计

结构一致性

当前交易方向是否顺着主趋势。逆势单会自动降级处理。

形态完整性

信号是否有前后结构支撑，而不是单根 K 线孤立出现。

Session Filter

选择当前时段。重叠时段质量较高，亚洲盘默认偏保守。

ATR 波动过滤

波动过低代表没量，过高代表过热，正常波动才是较优环境。

执行审计区（v2）关键项失败时，系统直接 BLOCK

Trigger Audit

是否真的出现明确信号，例如强阳、强阴、拒绝或吞没。

Key Zone Audit

是否位于关键区域，例如 Fibo、支撑阻力、回踩区、突破确认区。

Trap Zone Audit

是否接近高密度成交区、明显阻力位或容易诱多诱空的陷阱区域。
